# **App Name**: GDG SPECM Website

## Core Features:

- About Page: Comprehensive information about GDG and Shree Parekh Engineering College, Mahuva.
- Team Page: Showcase team members with roles, bios, and social links, including admin management.
- Student Progress Leaderboard: Real-time student progress tracking and display with filtering and search.
- Admin File Upload: Admin interface to upload CSV/Excel files for automatic leaderboard updates.
- Real-time Leaderboard Sync: Leaderboard automatically updates instantly when admin uploads a new file.
- Secure Admin Panel: Firebase Authentication protected admin panel.
- File Processing: Automatic CSV/Excel parsing and validation using Papaparse or SheetJS.
- Responsive Design: Fully responsive design for all devices (mobile, tablet, desktop).

## Style Guidelines:

- Primary color: Google Blue (#4285F4) for main branding elements.
- Accents: Red (#EA4335), Yellow (#FBBC04), Green (#34A853) for highlights.
- Background: White/Light Gray (#F8F9FA) for a clean look.
- Body font: Modern sans-serif font for readability.
- Headline font: Bold sans-serif font for headings.
- Lucide React icons: Simple and consistent icons for various sections.
- Mobile-first approach with a responsive grid layout.
- Smooth animations and transitions for a modern and engaging user experience.
